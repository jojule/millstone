- Minor bugfixes in Web Adpater packages.

version 3.1.0
-------------

Base:
- Added interface Field implemented by the AbstractField and its subclasses
- Added interface FieldFactory for creating Fields to Table and Form
- Added FormField component for recursive forms.
- Added Window-wide focus handling mechanism for Fields.
- Added  Component.Focusable interface.
- Added required property to all Fields.
- Added Protected setInternalValue function.
- Transaction listener interface to application context.
- Paint caching support to PaintTarget and implemented in AbstractComponent 
  and CustomComponent.
- Added possibility to get/iterate applications in application context.
- Added BeanItem constructor for limited set of properties.
- Added sortable property to Container.
- IndexedContainer: Support for Boolean comparisons.
- Added IndexedContainer Collection constructor to create a Container from 
  a collection of ItemIds.
- IndexedContainer implements Sortable and Cloneable
- Added java.lang.Comparable support to Label.
- New RAW output mode added to Label.
- Added sorting support to Table
- Added support for user selectable column headers and column header sorting 
  to Table.
- Added default theme support for user rearrangable (drag and drop) table columns and 
  support to hide (collapse) and show (expand) columns visually.
- Added paint caching support to allow client-side caching.
- PropertysetItem: cloneable support.
- Support disabled tabs in TabSheet
- New error handling framework for adapters and applications.
- Added TextField format-property to allow custom formatting of values.
- Upload implements Field and Component.Focusable interfaces.
- Added child component getter function to CustomLayout.
- FilesystemContainer now sorts the files by name.
- Added default constructor to all Ui components.
- Changed type checks to use instanceof operatoe instead of assignableFrom call.
- Promoted the immediate property to super interface VariableOwner.
- Table cell value formatting can now be overriden.
- GridLayout.removeAllComponents now also resets the cursor.
- Panel.removeAllComponents now calls respective method in layout.
- Changed the Property.ConversionException constructor argument type 
  from Exception to Throwable.
- PropertysetItem now maintains now the addition order of properties.
- Added Action.Container declaration to Tree.
- Caption property in Select can now be assigned be before datasource.
- Added UID:s for inner-classes as suggested by Eclipse IDE
- Fixed several smaller issues not listed here.

Web-adapter:
- WebAdapter support for new XSL/T  engines.
- Reorganized the WebAdapter default theme javascripts and added support for new UI component features.
- Several small fixes in WebAdapter default theme and adapter.


version 3.0.3
-------------
Base:
  - Resolved problem with update in ContainerHierarchicalWrapper (#623)
  - Fixed the recursive expand and collapse functions (#630, #622)
  - Propagate the Form attach and detach events to sub components (#632)
  - Parent is now assigned to Form layout (#632)
  - Resolved a NPE problem in object construction (#634)
  - Set Tabsheet immediate-property to true by default

WebAdapter:
  - Added support for themes in unpacked WAR
  - Default theme JAR is now bound to WebAdapter version
  - Fixed the problem with theme resources when theme
    extends an already inherited theme
  - Added possibility to specify defaults for WebAdapter parameters in
    system properties using 'org.millstone.webadapter.*' namespace.
    Following parameters are supported:
     + Debug - true|false, WebAdapter debug mode
     + DefaultThemeJar - string, Location default theme JAR 
     + ThemeSource - string, The external theme sources 
     + ThemeCacheTime - int, Cache expire time in seconds for theme resources
     + MaxTransformers - int, Number of concurrent transformers
     + TransformerCacheTime - int, Transformer expire time in seconds
  - Resolved problem with window closing itself (#618)
  - Adding handling for removed or invisible application windows
  - Null value changes are no longer ignored in HttpVariableMap (#635)
  - Description file is now always reloaded when reloading theme XSL
  - Theme description parsing exceptions are now fully printed
  - Updated debug information in theme loading
  
Default Theme:
  - Resolved link and button captions in Table (#633)
  - Added image width and height parameters to tree template.

Examples:
  - Added Tree component expandAll and collapseAll feature demos

version 3.0.1
-------------

Base:
