version 3.0.2
---------------------------
Base:
  - Fixed FrameWindow paint to use parent paint.
  - The table default for item caption mode is now EXPLICIT_DEFAULTS_ID as 
    documented before.
  - Removed the false unsupported validator exception in DateField.
  - Removed possibly duplicate value,item and property change events in Select.
  - The vertical scroll variable was painted regardless of scrollable property.
  - Added toString function and possibility to iterate the error messages 
    in CompositeErrorMessage.
  - Updated documentation.

WebAdapter:
  - Rewrote the XML escape functions.
  - Application locale now defaults to browser locale.
  - HttpVariableMap now ignores non-changed values.
  - Theme requirement rules are now correctly applied to filesets.
  - Fixed an extra variable change in Tabsheet.
  - Removed the unreliable CSS and frame checks
  - Updated documentation.
  - Updated browser probe HTML versions.
  - Added JScript 5.1 indentification
  - Set the DebugWindow default dimensions.
  - Removed XSL transformer debug

Default Theme:
  - Added missing width and height handling to Window.
  - The icon is now an active in Link component.
  - Removed border in button and link icons.
  - Added support for icon-only button.

Examples:
  - Disabled the window name property write when window is 
    added to application.
  - Updated StonePlayer example. 


version 3.0.1
---------------------------

Base:
- Removed unused import statements from several classes under base package.
- Table: Fixed td-sections to conform UIDL sepcification.
- Added missing caching parameters to all applicationResources

WebAdapter:
- Added support for different XSL/T processors to the Web Adapter.
  The XSL/T processor to be used is selected based on system properties.
  The Web Adapeter has been tested with J2SDK1.4.1(Xalan) and 
  Saxon 6.5.2 and 7.3.
- Fixed potential NullPointerException in WebBrowser equality check.
- Removed unused import statements from several classes.

Default Theme:
- Added support for Saxon6 by removing $dhtml-variables from template 
  match-patterns
- DateField: Fixed invalid mode attributes in named templates
- Button, Link: Fixed immediate switch mode bug in non-javascript browsers
  - Fixed description and error related bugs
  - Added support for Saxon6 by removing $dhtml-variables from template 
    match-patterns

Examples:
- Chat: Added message history. Fixed bugs related to leaving chat.
- FeatureDateField: Locale selector is now statically initialized
- OrderedLayoutFeature: Added initial content.
- Removed unused import statements from several classes.
