Millstone ™ Example Applications

These examples provide a good tutorial to the features if Millstone. The source code for the examples is extensively commented and we tried to make them as readable as possible. Trade-off is that the samples are little bit longer than thay should be.

Samples try to present the associated features as clearly as possible. Feel free to cut n' paste the code from the samples to implement the same behaviour in your own application.

The javadoc API-documentation is also available in the distribution package.
